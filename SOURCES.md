# Источники и политика данных SetupLab

SetupLab — статическое PWA для GitHub Pages. Оно не копирует магазинные базы, не выполняет скрытый парсинг и не выдаёт ориентировочные значения за цены в реальном времени.

## Каталог

В версии 1.3.0 каталог содержит 731 позицию в шести направлениях. Названия моделей и основные семейства сверяются по официальным линейкам производителей. Характеристики, цены, баллы, апгрейдность и описания доступны для локального редактирования.

## Ссылки на покупку

Для позиции формируются ссылки на поиск точного названия модели:

- Яндекс Маркет;
- Ozon;
- DNS для поддерживаемых категорий;
- официальный сайт или официальный поиск производителя.

Поисковые ссылки используются потому, что прямые карточки продавцов, наличие и адреса страниц регулярно меняются. Финальную цену, состояние товара, комплектацию, гарантию и условия доставки пользователь проверяет у продавца.

## Базовые цены

Все расчёты хранятся в EUR. В интерфейсе различаются:

- **проверенная цена** — отдельно пересмотренный ориентир с датой;
- **ориентировочная цена** — стартовое редактируемое значение;
- **локальная цена** — значение, изменённое пользователем на конкретном устройстве.

GitHub Pages не имеет серверного процесса для автоматического обновления 731 цены и не может надёжно обходить CORS и защиту интернет-магазинов.

## Баллы и апгрейдность

Баллы SetupLab предназначены для относительного сравнения внутри типа товара и направления сборки. Апгрейдность учитывает модульность, жизненный цикл платформы, запас по характеристикам и совместимость. Эти оценки не являются сертифицированными лабораторными измерениями.

## Расходы в год

Годовая оценка включает:

1. электроэнергию по мощности товара или сборки, числу часов работы и локальному тарифу;
2. регулярные расходы из поля `futureAnnualEUR` — обслуживание, расходные материалы, подписки или типичные замены.

Пользователь может изменить тариф, время работы и значения конкретных компонентов.

## Изображения

Используется каскад:

1. пользовательская фотография;
2. сохранённое открыто лицензированное изображение;
3. веб-предпросмотр по названию модели, загружаемый только около области просмотра;
4. локальная офлайн-иллюстрация.

Одновременно приложение загружает не более двух внешних изображений, чтобы не перегружать Safari на iPhone.
