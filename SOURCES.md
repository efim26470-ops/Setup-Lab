# Источники структуры каталога

Расширенный каталог составлен по актуальным линейкам производителей и их официальным разделам продукции. SetupLab не копирует магазинные базы и не выдаёт ориентировочные цены за котировки в реальном времени.

Основные официальные каталоги, использованные для проверки существования продуктовых линеек:

- AMD Ryzen и Ryzen Threadripper;
- NVIDIA GeForce RTX;
- MOZA Racing, Fanatec, Simagic, Asetek SimSports;
- LG, Samsung, Sony, Denon, Marantz;
- Dell UltraSharp, Logitech MX;
- Sony Alpha, Canon EOS R, Nikon Z, Fujifilm X/GFX;
- KEF, Yamaha, Marantz, Genelec, Neumann и другие производители аудиотехники.

Фотографии загружаются отдельно через Openverse. На карточке сохраняются автор и лицензия найденного изображения. Для конкретной позиции пользователь может открыть поиск фотографии и выбрать другой результат либо указать собственный URL.
