# Источники и политика данных SetupLab

SetupLab — статическое PWA для GitHub Pages. Оно не копирует магазинные базы, не выполняет скрытый парсинг и не выдаёт ориентировочные значения за цену в реальном времени.

## Каталог и официальные страницы

Существование моделей и основные характеристики сверяются по официальным линейкам производителей: AMD, Intel, NVIDIA, ASUS, MSI, MOZA Racing, Fanatec, Simagic, LG, Samsung, Sony, Denon, Canon, Nikon, Fujifilm, Apple, Dell, KEF, Yamaha, Genelec, Focusrite и другим брендам из каталога.

Кнопка «Официальный сайт» строит поиск по точному названию модели с ограничением на официальный домен бренда. Это позволяет находить международную страницу модели даже тогда, когда товар отсутствует у российских продавцов.

## Ссылки на покупку в России

Для каждой из 541 позиций формируются точные поисковые ссылки:

- Яндекс Маркет;
- Ozon;
- DNS для поддерживаемых категорий;
- официальный сайт производителя.

Поисковая ссылка используется намеренно: карточки продавцов, URL и наличие часто меняются, тогда как поиск по бренду и модели остаётся устойчивым. Финальную цену, комплектацию, гарантию и доступность пользователь проверяет на странице магазина.

## Базовые цены

Все расчёты хранятся в EUR. В карточках используются два статуса:

- **Цена проверена** — цена была пересмотрена по актуальному рыночному или официальному ориентиру, дата указана в карточке;
- **Ориентировочная цена** — редактируемая стартовая оценка, которую следует сверить по кнопке «Купить».

В версии 1.2.0 отдельно пересмотрены наиболее заметно завышенные позиции AMD Ryzen, MOZA R9, LG OLED C5, Apple Studio Display, Canon EOS R5 Mark II и другие популярные модели. У проверенных позиций может быть поле `priceReferenceUrl`.

## Валюты

Проверенные ориентиры на 10.07.2026 хранятся как количество единиц валюты за 1 EUR:

- EUR: 1;
- RUB: 86.5906;
- USD: 1.1435;
- GBP: 0.8518;
- CNY: 7.7712;
- RSD: 117.37.

Курсы остаются локально редактируемыми. SetupLab не обещает автоматическое обновление: GitHub Pages не имеет серверного процесса, а браузерные запросы к финансовым и магазинным сервисам могут блокироваться CORS или лимитами API.

## Изображения

Для изображения используется каскад:

1. пользовательская фотография;
2. сохранённая фотография Openverse;
3. веб-предпросмотр по точному названию модели;
4. локальная офлайн-иллюстрация.

При выборе изображения через Openverse приложение сохраняет автора и обозначение лицензии.
