# SetupLab changelog

## 1.2.0

- Быстрый выбор EUR, RUB, USD, GBP, CNY и RSD в верхней панели.
- Обновлены проверенные курсы на 10.07.2026.
- Для всех 541 компонентов добавлены ссылки Яндекс Маркет, Ozon, DNS и официальный поиск.
- Пересмотрены завышенные цены популярных AMD Ryzen и MOZA R9.

## 1.1.1
- Исправлена пустая область фотографий в каталоге.
- Добавлен прямой веб-предпросмотр изображения по названию модели без сотен запросов к Openverse API.
- Добавлен каскад: пользовательское/сохранённое фото → веб-фото → локальная офлайн-иллюстрация.
- Изображения теперь показываются целиком и не обрезаются через object-fit: contain.
- Локальные иллюстрации включены в PWA-кэш.
