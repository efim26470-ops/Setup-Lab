# SetupLab PWA

Статическое PWA-приложение для сравнения конфигураций ПК, автосима, домашнего кинотеатра, рабочего места, фотооборудования и музыкальных систем.

## Возможности

- расширенный стартовый каталог: 541 позиция в шести направлениях;
- несколько независимых сборок;
- редактируемый каталог `catalog.json`;
- пользовательские компоненты и импорт JSON;
- стоимость, условный балл производительности, цена балла;
- правила совместимости для разных типов комплектов;
- будущие расходы, электроэнергия и стоимость владения за 3 года;
- оценка апгрейдности и сравнительная диаграмма;
- темы: тёмная, светлая, Assetto Corsa;
- автоматическая ленивая загрузка веб-фотографий по точному названию модели;
- каскадный резерв: сохранённая фотография → веб-предпросмотр → локальная офлайн-иллюстрация;
- локальное хранение, экспорт и импорт резервных копий;
- офлайн-режим через Service Worker;
- адаптация под iPhone, safe-area и установку на экран «Домой».

## Публикация на GitHub Pages

1. Загрузите содержимое архива в корень нового репозитория.
2. Откройте **Settings → Pages**.
3. В разделе **Build and deployment** выберите **Deploy from a branch**.
4. Выберите ветку `main` и папку `/ (root)`.
5. Откройте выданный адрес GitHub Pages один раз с интернетом.

## Каталог

541 базовая позиция распределена между ПК, автосимом, домашним кинотеатром, рабочим местом, фотооборудованием и аудиосистемами. Базовые цены указаны в EUR как стартовые редактируемые оценки на дату, указанную в `catalog.json`. Это не магазин и не источник котировок в реальном времени. Изменить позицию можно:

- непосредственно в `catalog.json`;
- через карточку товара в приложении — изменение сохранится локально;
- импортом собственного JSON;
- добавлением пользовательской позиции.

Минимальный формат позиции:

```json
{
  "id": "unique-id",
  "group": "pc",
  "type": "gpu",
  "brand": "Brand",
  "name": "Model",
  "priceEUR": 500,
  "score": 80,
  "upgrade": 75,
  "futureAnnualEUR": 10,
  "powerW": 250,
  "specs": {"VRAM": "16 GB"},
  "compatibility": {},
  "image": "https://example.com/photo.jpg",
  "imageQuery": "Brand Model product photo"
}
```

## Важное ограничение

GitHub Pages — статический хостинг. Универсального открытого API со всеми моделями, актуальными ценами и спецификациями для шести разных рынков нет. Поэтому SetupLab использует локальный каталог, ручной импорт и каскад изображений. Карточка сначала пытается показать фотографию по точному названию модели, а при блокировке внешнего сервера автоматически переключается на локальную иллюстрацию из PWA-кэша. Ручной поиск открыто лицензированных изображений Openverse также сохранён в карточке компонента.
