# SetupLab PWA 1.3.1

SetupLab — статическое PWA для создания и сравнения конфигураций ПК, автосима, домашнего кинотеатра, рабочего места, фотооборудования и музыкальных систем. Проект рассчитан на GitHub Pages и хранит пользовательские данные локально.

## Что входит в версию 1.3.1

- 731 позиция в редактируемом `catalog.json`;
- ПК — 143 позиции;
- автосим — 113;
- домашний кинотеатр — 114;
- рабочее место — 117;
- фотооборудование — 116;
- музыкальная система — 128;
- краткое русскоязычное описание у каждой позиции;
- несколько независимых сборок, копирование и сравнение;
- стоимость, условный балл, цена балла, совместимость и полная стоимость владения;
- раздельный расчёт электроэнергии и регулярного обслуживания;
- подробное объяснение годовых расходов и апгрейдности;
- выбор EUR, RUB, USD, GBP, CNY и RSD;
- ссылки на поиск товара в Яндекс Маркете, Ozon, DNS и на официальном сайте;
- импорт каталога из JSON, пользовательские позиции и локальные исправления;
- семь тем: Dark, Light, Graphite, Midnight Navy, Titanium, Monochrome и Assetto Corsa;
- офлайн-режим, PWA-манифест, иконки и адаптация под safe area iPhone.

## Исправление стабильности Safari

Каталог больше не создаёт сотни тяжёлых карточек при запуске приложения. Он отрисовывается только после открытия вкладки и загружается порциями:

- 18 карточек на телефоне;
- 36 карточек на компьютере;
- следующие позиции добавляются кнопкой «Показать ещё»;
- внешние изображения загружаются только рядом с областью просмотра;
- одновременно выполняется не более двух запросов изображений;
- на мобильных устройствах отключены наиболее тяжёлые эффекты размытия и фоновые слои;
- предусмотрен резерв для Safari без `structuredClone`.

Эта схема снижает пиковое потребление оперативной и графической памяти при открытии каталога.

## Публикация на GitHub Pages

1. Полностью замените содержимое репозитория файлами из архива.
2. Откройте **Settings → Pages**.
3. Выберите **Deploy from a branch**.
4. Укажите ветку `main` и папку `/ (root)`.
5. Дождитесь завершения публикации.
6. На iPhone закройте старую вкладку SetupLab и откройте сайт повторно.

Если Safari продолжает показывать старую версию, удалите данные сайта в настройках Safari либо один раз откройте адрес с параметром `?v=1.3.1`.

## Как считаются показатели

**Расходы в год** состоят из двух частей: ориентировочной стоимости электроэнергии и регулярных расходов, сохранённых в поле `futureAnnualEUR`. Тариф и среднее количество часов работы в день меняются в настройках.

**Апгрейдность** — условная оценка от 0 до 100. Она учитывает запас платформы, модульность, доступность совместимых дополнений, резерв мощности и ограничения конкретной категории. В неполной сборке итоговая оценка дополнительно корректируется.

**Баллы** служат для сравнения моделей одного типа и сборок одного направления. Это прозрачная внутренняя модель SetupLab, а не единый лабораторный бенчмарк для несопоставимых категорий.

## Ограничения статического проекта

GitHub Pages не выполняет серверный парсинг магазинов. Базовые цены остаются редактируемыми ориентирами, а кнопки покупки ведут к актуальному поиску модели. Перед заказом следует проверить цену, комплектацию, гарантию и наличие у продавца.
