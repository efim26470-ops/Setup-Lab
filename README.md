# SetupLab PWA 1.4.0

SetupLab — статическое PWA для инженерного подбора, сравнения и учёта конфигураций ПК, автосима, домашнего кинотеатра, рабочего места, фотооборудования и музыкальных систем. Проект рассчитан на GitHub Pages: сервер и аккаунт не требуются, пользовательские данные хранятся локально.

## Каталог

- 764 позиции в редактируемом `catalog.json`;
- ПК — 143 позиции;
- автосим — 146;
- домашний кинотеатр — 114;
- рабочее место — 117;
- фотооборудование — 116;
- музыкальная система — 128;
- 55 позиций MOZA Racing, включая KS Pro, CS Pro, R21 Ultra, R25 Ultra, SRP2 и аксессуары;
- краткое русскоязычное описание у каждой позиции;
- ссылки на Яндекс Маркет, Ozon, DNS и официальный сайт производителя.

## Новые возможности версии 1.4

### Умный мастер подбора

Пользователь задаёт направление, бюджет, уровень, приоритет, страну покупки и предпочтительный бренд. Для автосима дополнительно учитываются платформа, дисциплина и способ установки. Мастер создаёт три сценария:

- экономный;
- сбалансированный;
- максимальный.

Алгоритм учитывает обязательные типы компонентов, внутренние баллы, цену, апгрейдность и совместимость.

### Глубокая совместимость

Проверки разделены на критические конфликты, предупреждения и информационные рекомендации. Для каждого замечания выводится рекомендуемое действие. Дополнительно анализируются:

- платформы PC/Xbox/PlayStation;
- экосистемы рулевых баз и рулей;
- момент базы и предел кокпита;
- форм-фактор дисциплины автосима;
- баланс CPU/GPU;
- проектор и экран;
- сокеты, память, питание, размеры, байонеты и импеданс.

### Sim Racing Lab

Отдельный модуль содержит:

- карту экосистем;
- сравнение рулевых баз по Н·м;
- подбор по дисциплине GT, Formula, Rally, Drift или Truck;
- проверку нагрузки на кокпит;
- рекомендации по бренду и платформе;
- последовательный путь апгрейда;
- создание рекомендуемой сборки в один клик.

### История цен

SetupLab не создаёт выдуманные исторические точки. Локальная история начинает накапливаться при просмотре, импорте или ручном изменении цены. В карточке товара отображаются:

- дата снимка;
- цена в EUR и выбранной валюте;
- изменение между первым и последним сохранённым значением;
- локальный график.

### Многокритериальный профиль

Помимо единого балла доступны десять осей:

- производительность;
- качество;
- надёжность;
- функциональность;
- совместимость;
- апгрейдность;
- цена/возможности;
- ликвидность;
- эргономика;
- тишина.

Часть осей является внутренней модельной оценкой и не заменяет профильные лабораторные измерения.

### Анализ слабых мест и следующий апгрейд

Для каждой сборки SetupLab показывает:

- критические конфликты;
- отсутствующие обязательные компоненты;
- компоненты ниже среднего уровня своего типа;
- слабую ось многокритериального профиля;
- до трёх совместимых вариантов следующего апгрейда;
- ориентировочную доплату с учётом перепродажи старого компонента за 55% его каталожной цены.

### Готовые комплекты

Добавлены пресеты для ПК, автосима, кинотеатра, рабочего места, фото и аудио. Каждый комплект создаётся как обычная локальная сборка и полностью редактируется.

### Общие ссылки и QR

Сборка кодируется в URL-фрагменте и может быть передана без сервера и аккаунта. Доступны:

- копирование ссылки;
- системное меню «Поделиться»;
- QR-код;
- просмотр и импорт полученной сборки.

QR-код загружается через внешний сервис, но данные сборки находятся непосредственно в ссылке.

### Мои устройства

Локальная база владения хранит:

- статус: куплено, заказано, планируется, ремонт или продано;
- дату и цену покупки;
- магазин;
- гарантию;
- серийный номер;
- ссылку на чек;
- заметки и историю обслуживания.

Компоненты можно добавить по одному из каталога либо целиком из сохранённой сборки.

- 18 карточек на телефоне и 36 на компьютере;
- изображения загружаются рядом с областью просмотра;

- тяжёлые эффекты отключаются на мобильных устройствах;
- новые инженерные разделы рендерятся только при открытии;
- предусмотрено восстановление технического кэша.

## Публикация на GitHub Pages

1. Полностью замените содержимое репозитория файлами из архива.
2. Откройте **Settings → Pages**.
3. Выберите **Deploy from a branch**.
4. Укажите ветку `main` и папку `/ (root)`.
5. Дождитесь завершения публикации.
6. На iPhone закройте старую вкладку SetupLab и откройте сайт повторно.

Если Safari показывает старую версию, один раз откройте сайт с параметром `?v=1.4.0` либо используйте кнопку локального восстановления на аварийном экране.

## Файлы проекта

- `index.html` — оболочка интерфейса;
- `styles.css` — адаптивный дизайн и темы;
- `app.js` — базовый каталог, сборки, расчёты и PWA;
- `features.js` — инженерные модули версии 1.4;
- `catalog.json` — полный редактируемый каталог;
- `catalog.min.json` — облегчённая копия для быстрого запуска;
- `sw.js` — офлайн-кэш;
- `manifest.webmanifest` — PWA-манифест.

## Ограничения статического проекта

GitHub Pages не выполняет серверный парсинг магазинов. Цены остаются редактируемыми ориентирами, а история накапливается локально. Перед покупкой необходимо проверить комплектацию, гарантию, совместимость, стоимость доставки и наличие у продавца.
